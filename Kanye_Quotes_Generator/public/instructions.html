Kanye West Quote Generator

Goal

Retrieve information from an API and display a new quote every 30 seconds.

Level 1

  1. Retrieve a quote from the API
  2. Change the quote displayed every 30 seconds

Level 2

  1. When you load a new quote, save the previous one, and display a "Previous x quotes" section with the current accumulated quotes.
  2. Limit the previous quotes displayed to 10. They should be shown in the order in which they were loaded (eg: last one on top of the previous quotes)

Level 3

  1. When no quote is loaded or when a quote is loading, display a "Loading..." message, or a spinner
  2. If there was an error fetching a quote; display an error message to the user (eg: "There was a problem loading a Kanye quote, please try again later")

In this exercise, you can use Fetch or Axios (or why not practice with both? ^^).
